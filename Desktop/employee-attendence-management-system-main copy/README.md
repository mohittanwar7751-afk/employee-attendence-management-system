  # Industry-Based Employee Attendance Management Web Application

## Description
A simple web-based application developed using Python and web technologies
to manage employee attendance for small and medium-scale industries.

## Technologies Used
- Python
- Flask (Basic)
- HTML
- CSS
- JavaScript
- SQLite
- GitHub

## Features
- Employee login
- Online attendance marking
- Admin dashboard
- Attendance reports

## How to Run
1. Install Python
2. Install Flask: pip install flask
3. Run app.py
4. Open browser and go to http://127.0.0.1:5000/
